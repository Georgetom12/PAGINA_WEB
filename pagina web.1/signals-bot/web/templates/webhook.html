Formato de la alerta en TradingView

En TradingView → Alertas → Crear alerta → pestaña "Notificaciones" → activa "Webhook URL".
En el campo "Mensaje" de la alerta pon este JSON: 

{
  "secret":   "{{ webhook_url.split('secret=')[1] if 'secret=' in webhook_url else 'TU_SECRETO' }}",
  "symbol":   "{{"{{"}}ticker{{"}}"}}",
  "action":   "BUY",
  "price":    {{"{{"}}close{{"}}"}},
  "interval": "{{"{{"}}interval{{"}}"}}",
  "exchange": "{{"{{"}}exchange{{"}}"}}",
  "message":  "Texto libre de tu alerta"
}

💡 Cambia "BUY" por: SELL, STRONG_BUY, STRONG_SELL, LONG, SHORT, CLOSE, ALERT

Formato alternativo (texto simple)

Si prefieres texto plano en vez de JSON, usa este formato separado por |:

{{ webhook_url.split('secret=')[1] if 'secret=' in webhook_url else 'TU_SECRETO' }}|BUY|{{"{{"}}ticker{{"}}"}}|{{"{{"}}close{{"}}"}}|{{"{{"}}interval{{"}}"}}|Descripción opcional
