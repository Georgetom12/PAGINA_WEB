🐋 WHALE FLOW PRO — Cómo llenar cada campo
🐋 Ballenas
(-100 a +100)
Copia el valor "Ballenas" directamente de la tabla del indicador en TradingView.
Positivo (ej: 0.8, 30, 80) = ballenas comprando.
Negativo (ej: -20, -60) = ballenas vendiendo.
Ejemplo tuyo con DOGE: el indicador muestra 0.8 → pon 0.8
🫙 Manos Débiles
Copia el valor "Manos Débiles" o "Retail" de la tabla del indicador.
Valor alto (>60) con ballenas bajas = señal débil, retail dominando.
Valor bajo (<30) = retail saliendo, posible acumulación institucional.
Ejemplo tuyo: indicador muestra 51.3 → pon 51.3
📊 RVOL (x)
multiplicador
Es el Relative Volume (volumen relativo). Cópialo de la tabla del indicador.
>2.0x = volumen institucional alto (ballenas activas).
1.0-2.0x = volumen normal.
<0.5x = volumen muy bajo, no hay actividad.
Ejemplo tuyo: indicador muestra 0.87x → pon 0.87
💧 Flujo Alcista (%)
Si el indicador muestra el % directamente, cópialo. Si no:
SituaciónValor a poner
Ballenas >30 y precio sube con volumen alto60-80%
Ballenas neutral (0 a 10) o RVOL normal40-55%
Ballenas negativas o RETAIL DOMINA20-40%
Ballenas muy negativas + RVOL alto bajista5-20%
Ejemplo tuyo (DOGE): Ballenas=0.8, RVOL=0.87x, Retail domina → pon 45%

El indicador diceBotón a presionar

💡 Regla simple: Si el precio bajó a barrer mínimos y rebotó = LONGS cazados (alcista). Si subió a barrer máximos y cayó = SHORTS cazados (bajista).

📈 EMA 50 Bias
¿El precio está por encima o por debajo de la EMA 50?
SOBRE = precio > EMA 50 → bias alcista.
BAJO = precio < EMA 50 → bias bajista.
Lo puedes ver directamente en el gráfico si tienes la EMA 50 activada.
◆ POC vs Precio
El POC (Point of Control) es el nivel de precio con mayor volumen negociado (línea del Volume Profile):
SOBRE POC = precio > POC → comprado a precio justo o más caro.
BAJO POC = precio < POC → barato respecto al valor.
EN POC = precio exactamente en el POC → zona de equilibrio.
⚡ Liq. Barrida
¿El precio acaba de barrer una zona de liquidez y rebotó?
BARRIDA = se barrió liquidez recientemente (stop hunt) y el precio revirtió. Es una señal fuerte.
─ = sin barrida reciente.

🌊 Esquema Wyckoff
¿El rango actual es de acumulación o distribución?
ACUMULACIÓN = las instituciones están comprando (rango antes de subida).
DISTRIBUCIÓN = las instituciones están vendiendo (rango antes de caída).
Lo puedes ver por el contexto: si venía de una bajada larga = probable acumulación. Si venía de una subida larga = probable distribución.
💡 Combinación más poderosa: FASE C + SPRING + SSL CAZADO + Score Bull > 60% = señal de entrada Wyckoff de máxima calidad.
